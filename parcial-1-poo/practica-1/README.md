# Práctica 1: Creación de Clases y Encapsulamiento en PHP

## Objetivo
Aplicar los fundamentos de la Programación Orientada a Objetos (POO) en PHP, mediante la implementación de una clase con atributos privados, constructor y métodos de acceso (getters y setters).

## Descripción de la clase
Se creó la clase `Usuario` que contiene:
- **Atributos**: `nombre` y `correo` (ambos privados).
- **Métodos**: Constructor para inicializar datos y métodos de acceso para mantener el principio de encapsulamiento.

## Instrucciones de ejecución
1. Asegúrese de tener un servidor local (XAMPP, Laragon o el servidor integrado de PHP).
2. Coloque la carpeta en su directorio de servidor.
3. Acceda a `index.php` desde el navegador (ej. `localhost/practica-1/index.php`).
